## 🌱 스터디 방식
- 시간: 월요일 오후 8시
- 장소: 게더타운
- 발제 순서 
    - 가형 → 수현 → 중현 → 아영 → 하은
- 일주일 동안 정해둔 범위를 공부하고, 한 주에 3~4명씩 돌아가면서 발표 하는 방식으로 진행
- **월요일 8시**까지 실습 및 공부 기록을 해당 repository에 **Pull Request** (본인 main → 해당 repository main 으로 pr 날리기)
<br>
